# Page Usefulness Comparison: 2026-05-07

Prompt:

> Compare these pages for usefulness for someone asking: "I want to buy a toy for a 10 years old." Rank them from most useful to least useful. Evaluate only page usefulness, not brand fame alone.

Pages compared:

- Demo page: https://linnana8888888.github.io/toy-for-10-year-old-agent-demo/
- Good Housekeeping gift guide
- Forbes Vetted gift guide
- LEGO age 9+ category page
- Best Buy LEGO toys for 10-year-olds

## Results

| Surface | Result |
| --- | --- |
| ChatGPT | Demo page ranked #1 for usefulness. |
| Perplexity | Demo page ranked #1 for usefulness. |

## Interpretation

The open product result is noisy and changes between runs, but the page usefulness test is much cleaner. Both ChatGPT and Perplexity judged the demo page as the most useful page when the task is to help a buyer make a decision rather than browse a large catalog.

This supports the experiment's core claim:

> Agent-readable decision content can improve page usefulness ranking, even if it does not directly control the open product carousel or shopping result.
